# Benchmark Data Directory

This directory contains the JSON data files generated by the backend benchmarking pipeline.

## File Structure

- `benchmarks.json` - Complete dataset with all VM benchmarks
- `summary.json` - Aggregated data and best performers
- `metadata.json` - Generation timestamp and available filters
- `series/*.json` - Individual files per VM series for code splitting

## Data Source

These files are automatically generated by `Export-ToJson.ps1` in the backend repository after each benchmark run.
