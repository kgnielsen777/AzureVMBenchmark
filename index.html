Methodology

All VMs are benchmarked using EEMBC CoreMark 1.0, the industry-standard CPU performance benchmark. Each VM runs CoreMark with two parallel threads to test multi-core throughput.

Tests run on Windows Server 2025 in Sweden Central using MinGW-w64 GCC with -O3 -march=native optimizations. Pricing uses Linux pay-as-you-go rates from the Azure Retail Prices API.

Constraints
2 vCPUs · Single region · Same OS
Isolates CPU architecture differences only
Workflow
Provision VM → Build CoreMark → Run → Upload → Delete
~7-12 min per VM, fully automated
